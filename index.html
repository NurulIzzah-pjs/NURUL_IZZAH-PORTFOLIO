Final Year Project - Projek Sedekah Jariah

This final year project is an outcome of inspiration from Projek Sedekah Jariah (Abd Ghani), an NGO handling charity events. It addresses their main issues which is long waiting times, congestion at checkpoints via RFID scanning (MQTT), and automates inventory management through a POS system. It also assists families in managing aid money responsibly.

  • IoT Integration
    • Automated aid distribution tracking
    • Real-time notifications for inventory
  • RFID Authentication
    • Secure access for registered users
    • Quick scan for attendance
  • POS System
    • Track stock levels in real-time
    • Alerts for low or expired items
    • Process transactions efficiently
    • Generate monthly finance reports
